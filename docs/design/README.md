# InterNull Design System

Enterprise-grade design system for institutional blockchain privacy infrastructure.

## Files

- `design-system.md` - Comprehensive design specification (PRIMARY)
- `quick-reference.md` - Quick lookup guide for colors, typography, components
- `color-palette-reference.html` - Visual color palette reference (open in browser)
- `component-examples.html` - Live component showcase (open in browser)
- `design-tokens.css` - CSS custom properties/variables

## Quick Start

1. Read `quick-reference.md` for fast lookups
2. Reference `design-system.md` for comprehensive specs
3. Open `component-examples.html` in browser for visual examples

## Design Principles

- Enterprise-grade professionalism
- Institutional trust and credibility
- Technical sophistication
- Regulatory confidence
- Modern restraint (no crypto stereotypes)
